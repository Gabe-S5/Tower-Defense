# Tower Defense!

__A classic tower defense game! Will add upgrades, textures, sounds, and maps in the future.__
Link [here](www.gabetd.tk)

## Keybinds
* All towers can be selected with their first letter
    * (Q)rrow Tower
    * (W)niper Tower
    * Etc..
* Z to sell tower
* Esc to remove hovering information
